# GoIM 问题反馈记录

## 使用规则

- 每发现一个问题，新增一行；问题修复后保留记录并更新状态，不删除。
- 严重级别：`P0` 服务不可用或数据严重损坏；`P1` 核心功能不可用；`P2` 功能异常但有绕行方案；`P3` 体验、文档或测试问题。
- 状态：`待确认`、`待修复`、`修复中`、`待回归`、`已关闭`、`不修复`。
- “复现步骤”需包含账户、请求、响应和时间，敏感 Token 不写入文档。
